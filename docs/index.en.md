# Welcome

Welcome to the tutorial for the **member publications module** of the Centre d'études nordiques (CEN) intranet.

This guide walks you step by step through the publications module: adding and editing publications, importing from external sources, exporting in various formats, and searching the existing catalogue.

---

## Getting started

If this is your first visit, we recommend starting with **Create and edit publications**, then following the guides in order.

> This tutorial is intended for CEN members with intranet access. If you do not yet have an account, please contact the administrator.
