# Accéder au module de publications

## Connexion à l'intranet

Pour accéder au module de publications, vous devez d'abord vous connecter à l'intranet du CEN avec votre adresse courriel et votre mot de passe.

<figure markdown>
  ![Page de connexion de l'intranet](../assets/screenshots/login.png)
  <figcaption>Page de connexion de l'intranet CEN</figcaption>
</figure>

## Naviguer vers le module

Une fois connecté, repérez la carte du module des publications. Cliquez sur **Ouvrir** pour accéder au module.

<figure markdown>
  ![Menu principal de l'intranet](../assets/screenshots/nav-publications.png)
  <figcaption>Menu principal — cliquez sur « Ouvrir »</figcaption>
</figure>

## Page d'accueil du module

Vous arrivez sur la page principale du module de publications. C'est à partir d'ici que vous pouvez gérer l'ensemble de vos publications : les consulter, les ajouter, les modifier, les importer ou les exporter.

<figure markdown>
  ![Page d'accueil du module de publications](../assets/screenshots/publications-home.png)
  <figcaption>Page d'accueil du module de publications</figcaption>
</figure>
