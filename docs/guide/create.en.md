# Creating a Publication

You can add a new publication manually directly in the publications module.

## Accessing the creation form

From the module home page, click the **Add a publication** button.

<figure markdown>
  ![Add a publication button](../assets/screenshots/create-button.png)
  <figcaption>Click "Add a publication"</figcaption>
</figure>

## Filling in the form

Fill in the form fields with the publication's information. Required fields are marked with an asterisk (*).

<figure markdown>
  ![Creation form](../assets/screenshots/create-form.png)
  <figcaption>Publication creation form</figcaption>
</figure>

The main available fields are:

- **Title** * — full title of the publication
- **Authors** * — list of authors
- **Type** * — publication type (article, book, chapter, conference, etc.)
- **Year** * — publication year
- **Journal / Publisher** — name of the journal or publisher
- **DOI** — digital object identifier
- **Abstract** — publication abstract
- **Keywords** — associated keywords
- **URL** — link to the publication online

## Saving the publication

Once the form is complete, click **Save** to add the publication to the catalogue.

<figure markdown>
  ![Save button](../assets/screenshots/create-save.png)
  <figcaption>Click "Save" to confirm the creation</figcaption>
</figure>
